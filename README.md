# Power BI Dashboard Project

## Project Overview
This project contains an **interactive Power BI dashboard** designed to analyze and visualize business data.
The dashboard helps in tracking key metrics, identifying trends, and supporting data-driven decision making.

---

## Tools & Technologies Used
- Power BI Desktop
- DAX
- Data Modeling
- Excel / CSV Dataset
- GitHub

---


![Power BI Dashboard](images/dashboard_overview.png)

---

##  Key Features & Insights
- Interactive dashboards with slicers and filters
- KPI cards for quick performance overview
- Trend analysis using charts and graphs
- Clean and user-friendly layout
- DAX measures for calculated metrics

---

##  Files in This Repository
- `Visualizations.pbix` → Power BI dashboard file
- `images/` → Dashboard screenshots
- `README.md` → Project documentation

---

## How to Use This Project
1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Refresh data if required
4. Interact with slicers and visuals to explore insights
